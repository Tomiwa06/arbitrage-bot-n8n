# arbitrage-bot-n8n
Arbitrage Sentinel is an automated Node.js bot built with n8n. It scans Binance, Bybit &amp; OKX every 30s to find Cross-Exchange, Triangular, and P2P profit gaps. Unlike basic trackers, it factors in live gas fees, trading commissions, and network latency to deliver validated, actionable signals to Telegram. Precise, scalable, and fast.
